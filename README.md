# Проекты Яндекс.Практикум

### Здесь собраны все проекты, выполненные мной в процессе обучения на курсе "Специалист по Data Science". Каждый лежит в отдельной папке, которые размечены как "Тема_курса.Название_проекта"

### Внутри каждой папки лежат 4 типа файлов: ipynb-тетрадь, pdf-файл, html-документ и файл README.md с кратким описанием проекта и используемых библиотек 

### Данные проекты имели следующий порядок:

- Предобработка данных
- Исследовательский анализ данных
- Статистический анализ данных
- Сборный Проект — 1
- Введение в машинное обучение
- Обучение с учителем
- Машинное обучение в бизнесе
- Сборный Проект — 2
- Численные методы
- Временные ряды
- Машинное обучение для текстов
- Компьютерное зрение
