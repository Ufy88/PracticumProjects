# Проекты из Яндекс.Практикум

### Здесь собраны все проекты, выполненные мной в процессе обучения на курсе "Специалист по Data Science". Каждый лежит в отдельной папке, которые размечены как "Тема_курса.Название_проекта"

### Внутри каждой папки лежат 4 типа файлов: ipynb-тетрадь, pdf-файл, html-документ и zip-архив с .md файлом внутри. Для комфортного ознакомления с проектом без Jupyter Notebook я рекомендую html-документ, он лучше всего отображает тетрадь. 

### Данные проекты шли в следующем порядке:
- Предобработка данных
- Исследовательский анализ данных
- Статистический анализ данных
- Сборный Проект — 1
- Введение в машинное обучение
- Обучение с учителем
- Машинное обучение в бизнесе
- Сборный Проект — 2
- Численные методы
- Временные ряды
- Машинное обучение для текстов
- Компьютерное зрение
